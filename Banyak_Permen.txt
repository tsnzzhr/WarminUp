Seorang penjual permen didatangi 3 anak, masing-masing ingin membeli permen. Tiap anak ingin paling tidak mendapat satu permen. Jika mereka membeli B permen, berapa banyak cara bagi ketiga anak tersebut untuk membagi semua permen tersebut di antara mereka bertiga?

Input Format

Pada baris pertama, masukkan B.

Constraints

0 <= B <= 65536

Output Format

Output berupa banyak cara membagi permen di antara 3 anak tersebut diakhiri dengan endline.

Sample Input 1
3
Sample Output 1
1
Sample Input 2
4
Sample Output 2
3
